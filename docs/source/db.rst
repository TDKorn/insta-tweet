The ``db`` and ``models`` modules
-----------------------------------

The :mod:`~.db` and :mod:`~.models` modules are utilized when saving, loading, and InstaTweeting a :class:`~.Profile`
located on a remote, SQLAlchemy supported database

* The :mod:`~.db` module contains functions for interacting with your database
* The :mod:`~.models` module specifies the ``Profiles`` table that is used to store profile configurations (in the form of pickle bytes)
