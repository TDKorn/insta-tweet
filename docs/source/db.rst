The ``db`` module
-----------------------------------

The :mod:`~.db` module contains the :class:`~.DBConnection` class as well as the :class:`~.Profiles` database table

* Any database that can be connected to using ``SQLAlchemy`` is compatible with ``InstaTweet``
* Sessions are made/destroyed when saving, loading, and InstaTweeting a :class:`~.Profile`
